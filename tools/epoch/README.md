# epoch

时间戳 ↔ 日期双向转换。**自动识别**输入类型，一次输出多种格式，开发/排查日志时不再来回查公式。

## 用法

```bash
epoch                              # 现在
epoch 1714492800                   # 10 位 → Unix 秒
epoch 1714492800000                # 13 位 → Unix 毫秒
epoch 1714492800123456             # 16 位 → Unix 微秒
epoch 1714492800123456789          # 19 位 → Unix 纳秒
epoch '2024-04-30 15:30'           # 日期时间（按本地解释）
epoch '2024-04-30T15:30:00+08:00'  # ISO 8601 含时区
epoch -c                           # 从剪贴板读
echo 1714492800 | epoch            # 从管道读
```

## 输入识别

| 形式 | 解释 |
|---|---|
| 空 / `now` / `当前` / `现在` | 当前时间 |
| 10 位纯数字 | Unix 秒 |
| 13 位纯数字 | Unix 毫秒 |
| 16 位纯数字 | Unix 微秒 |
| 19 位纯数字 | Unix 纳秒 |
| 其他位数纯数字 | 按 Unix 秒解释（带提示） |
| `YYYY-MM-DDTHH:MM:SS±HH:MM` | ISO 8601 |
| `YYYY-MM-DD HH:MM[:SS]` / `YYYY/MM/DD ...` | 日期时间，按本地时区解释 |
| `YYYY-MM-DD` / `YYYYMMDD` | 日期 |

## 输出

```
输入: 1714492800  →  Unix 秒（10 位）

  Unix 秒    1714492800
  Unix 毫秒  1714492800000
  Unix 微秒  1714492800000000
  ISO 8601   2024-04-30T16:00:00Z
  UTC        2024-04-30 16:00:00
  北京时间   2024-05-01 00:00:00 CST
  本地时间   2024-05-01 00:00:00 CST
  RFC 2822   Tue, 30 Apr 2024 16:00:00 +0000
  相对       1 年前
```

终端会上色（key=青、value=绿、识别标签=黄）。设 `NO_COLOR=1` 关闭。
