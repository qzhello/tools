# sizex

目录大小可视化。比 `du -sh *` 直观：自动按大小排序、条形图、彩色梯度，一眼看出"谁吃了硬盘"。

## 用法

```bash
sizex                       # 当前目录一级条目，前 20 项
sizex ~/Downloads           # 指定路径
sizex -n 50                 # 前 50 项
sizex -a                    # 包含隐藏文件（. 开头）
sizex -f                    # 递归找最大的单个文件（不是目录）
sizex -f -n 10 ~/code       # 在 ~/code 下找最大的 10 个文件
sizex --no-bar              # 不画条形图，纯数字
sizex -j                    # JSON 输出
sizex /path/to/file         # 单文件直接报大小
```

颜色（按大小）：

| 大小       | 颜色 |
|-----------|------|
| ≥ 1 GB    | 红   |
| ≥ 100 MB  | 黄   |
| ≥ 1 MB    | 绿   |
| < 1 MB    | 灰   |

类型标记：`/` 蓝色 = 目录，`@` 青色 = 符号链接，无标记 = 文件。

## 选项

| 选项 | 说明 | 默认 |
|------|------|------|
| `-n, --top N`   | 显示前 N 项     | 20 |
| `-a, --all`     | 包含隐藏项      | 否 |
| `-f, --files`   | 递归找最大文件  | 否 |
| `-L, --follow`  | 跟随符号链接    | 否 |
| `--no-bar`      | 不画条形图      | 否 |
| `-j, --json`    | JSON 输出       | 否 |

## 行为说明

- 默认**不跟随符号链接**（避免重复计算 / 死循环）
- 权限不足的项会被静默跳过
- `-f` 模式扫描所有子文件，目录大时较慢；扫描 > 1s 会在末尾打印用时
